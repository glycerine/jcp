jcp: Jason's rsync-like copy
============================

`jcp` is an rsync-like library and command line tool written Go. 

It is up to 3x faster than regular C rsync.

installation
------------
~~~
$ go install github.com/glycerine/jcp/cmd/jcp@latest
~~~
